# Portfolio Website

My personal portfolio website showcasing my projects and skills.

**Live Site:** [dhirengithub.github.io/portfolio-website](https://dhirengithub.github.io/portfolio-website/)

## Tech Stack

- React 19
- TypeScript
- React Router
- Vite

## Features

- Home page with animated skill badges
- Projects showcase with detail pages
- Timeline section
- Contact page

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

## Deployment

Automatically deploys to GitHub Pages via GitHub Actions on push to `main`.
